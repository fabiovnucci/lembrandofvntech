## 📂 O que tem aqui?

### 🎨 Módulo CSS & Semântica (Ativo)
Guia visual sobre os pilares do Front-End moderno:
- **Box Model:** Entendendo Margins, Borders e Paddings na prática.
- **Flexbox:** Alinhamento de elementos de forma profissional e dinâmica.
- **Responsividade:** Estrutura adaptável para celulares e tablets.
- **Semântica HTML5:** Uso correto de tags como `<section>`, `<article>` e `<main>`.

### ⚙️ Módulo Git & GitHub (Ativo)
Fluxo de trabalho completo para versionamento:
- **Fluxo Inicial:** `git init`, `status` e `add`.
- **Versionamento:** `commit` e `push origin main`.
- **Resolução de Conflitos:** Comandos de emergência como `push --force`.
