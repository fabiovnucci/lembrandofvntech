01. O Modelo de Caixa (Box Model) Tag: <section>

Margin vs Border vs Padding

O conceito mais importante para não quebrar o layout.

  • Margin: Espaço externo (afasta vizinhos).
  • Border: A linha de contorno.
  • Padding: Espaço interno (respiro do conteúdo).
margin: 20px; border: 4px solid #333; padding: 30px;
CONTEÚDO

Vermelho = Margin | Azul/Verde = Padding

02. Alinhamento com Flexbox Tag: <article>

display: flex;

Como alinhar itens profissionalmente sem usar "floats".

  • justify-content: Alinha horizontalmente.
  • align-items: Alinha verticalmente.
  • gap: O espaçador moderno.
display: flex; justify-content: center; gap: 15px;
